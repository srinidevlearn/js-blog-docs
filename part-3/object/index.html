  • When an object is created with an object literal, any object variable derived from that object will act as a clone of the original object. Hence, any change you make in one object will also reflect in the other object.

Some Inbuilt JS Function
  • Object.assign() - copies enumerable props from one or more source objects to a target object, returns target objects
  • Object.keys() - extract enumerable keys from object and returns as array
  • Object.values() - extract value from object and returns as array
  • Object.entries() - array of a given object's own enumerable string-keyed property [key, value] pairs, in the same order as that provided by a for...in loop
  • Object.fromEntries() - reverse of object.entries ()
