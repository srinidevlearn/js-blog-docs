Map
  1. it is immutable
  2. creates a new array from calling a function for every array element.
  3. calls a function once for each element in an array.
  4. if element was empty it won't execute
Filter
  1. it is immutable
  2. creates a new array filled with elements that pass a test provided by a function.
  3. if element was empty it won't execute
  4. If no elements pass the test it returns an empty array.
Some
  1. it is immutable
  2. method checks if any array elements pass a test (provided as a callback function).
  3. method returns true (and stops) if the function returns true for one of the array elements.
  4. method returns false if the function returns false for all of the array elements.
  5. if element was empty it won't execute
Every
  1. it is immutable
  2. method checks if any array elements pass a test (provided as a callback function).
  3. method returns true if the function returns true for all elements.
  4. method returns false if the function returns false for one element.
  5. if element was empty it won't execute
Find
  1. it is immutable
  2. method returns the value of the first element that passes a test.
  3. method returns undefined if no elements are found.
  4. if element was empty it won't execute
Includes
  1. method returns true if an array contains a specified value
  2. method returns false if the value is not found.
  3. method is case sensitive.
Find Index
  1. it is immutable
  2. method returns the index (position) of the first element that passes a test.
  3. method returns -1 if no match is found.
  4. method does not execute the function for empty array elements.
Last Index
  1. method returns the index (position) of the first element that passes a test.
  2. method returns -1 if no match is found.
  3. By defalt the search starts at the last element and ends at the first (left to right)
Index Of
  1. method returns the index (position) of the first element that passes a test.
  2. method returns -1 if no match is found.
  3. By defalt the search starts at the last element and ends at the first (right to left)
Concat
  1. it is immutable
  2. method concatenates (joins) two or more arrays
  3. method returns a new array, containing the joined arrays.
Join
  1. joins array to string
Splice
  1. method adds and/or removes array elements.
  2. method overwrites the original array.
  3. It is Mutable
Slice
  1. returns selected elements in an array, as a new array.
  2. method does not overwrites the original array.
  3. It is immutable
