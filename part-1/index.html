JS Varaibles
  • A variable is a 
    “named storage”
    for data.
  • To create a variable in JavaScript, use the 
    let
    , 
    const
    keyword
  • We can also declare multiple variables in one line
Display method in JS
  • console.log()
  • alert()
  • document.write() // writes at entire document
  • innerHTML='' //writes inside the element
Datatypes in JS
  • number
    integer or floating-point, integers are limited by ±(253-1).
  • bigint
    bigint values — which are too large to be represented by the number primitive.
  • string
    A string may have zero or more characters, there’s no separate single-character type.
  • boolean
    true or false
  • null
    for unknown values
    a standalone type that has a single value null
  • null
    for unassigned values
    a standalone type that has a single value undefined.
  • object
    for more complex data structures
  • symbol
    can be used for unique identifiers
Variable Declaration vs variable definition:
  • A variable declared will be 
    let declaredVariable;
  • A variable defined will be 
    declaredVariable = true;
Variable Hoising
  • Variables defined with let and const will be hoisted to the top of the block, but not initialized.
  • Meaning: The block of code is aware of the variable, but it cannot be used until it has been declared.
  • If variable's were used before it is declared it will throw 
    ReferenceError
Usage of Strict
  • Keywords reserved for future JavaScript versions can NOT be used as variable names in strict mode.
    • implements
    • interface
    • let
    • package
    • private
    • protected
    • public
    • static
    • yield
  • If the object is not specified, functions in strict mode will return undefined and functions in normal mode will return the global object 
    window
Functions
  • Block of code to designed to perform specific task.
  • Function can be executed by calling it
  • It will have function keyword, name of the function and parentheses
  • Within parentheses we can pass parameter
  • Function execution will stop when it encounters return statement
